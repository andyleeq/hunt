## 二、因果链追踪

### 链条A：权力链

```
沈执父母掌握黑料
  → 挡住高天禄升迁
    → 高天禄授意灭门
      → 厂区改制成功
        → 高天禄升迁
          → 打压陆父
            → 陆父抑郁早逝
              → 陆岑成为刑警追查铁三角
                → 高天禄再次用权力封杀调查
```

### 链条B：资本链

```
沈执父母掌握厂区地块信息
  → 秦九鼎觊觎地块
    → 出钱雇凶灭门
      → 获得第一块黄金地
        → 建立资本帝国
          → 用小钱封口赵父
            → 赵家赤贫
              → 赵磊投靠刀疤荣
                → 成为铁三角白手套
```

### 链条C：暴力链

```
刀疤荣为上位机会
  → 亲手带人灭沈执父母门
    → 成为高·秦的暴力工具
      → 打残阻拦保安赵父
        → 登顶黑老大
          → 手下成为沈执的猎物
            → 手下不断失踪
              → 归因为"仇家寻仇"
```

### 链条D：秩序链

```
沈执父母被灭门
  → 10岁沈执被苏家救下
    → 掠食者本能觉醒
      → 确立秩序铁律
        → 考取法医副教授
          → 合法搜集铁三角罪证
            → 锁定亲手行凶者
              → 秘密清算（20年）
                → 铁三角完全不知
```

### 链条E：恩情链

```
灭门夜
  → 苏晚父母藏沈执进衣柜
    → 幼年苏晚用身体挡柜门
      → 恩情刻入沈执本能
        → 苏晚父母临终遗命
          → 苏晚与沈执契约结婚
            → 苏晚怀孕
              → 胎儿基因逆向异化苏晚
                → 沈执"受恩必还"铁律发动 → 保留苏晚
```

### 链条F：悲剧镜像链

```
灭门案同时产生两个受害者：
  ├── 沈执（父母被杀）→ 走向秩序清算
  └── 赵磊（父亲被打残）→ 走向以恶救父
       ↓
  两人命运平行20年
       ↓
  赵磊"亲手"伤害苏晚
       ↓
  进入沈执清算名单
       ↓
  受害者清算受害者 = 命运最残忍的闭环
```

## 三、信息差矩阵

> 此矩阵记录每个角色"知道什么"和"不知道什么"，是写作时避免逻辑漏洞的核心工具。

### 沈执的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 灭门案全部真相 | —（他是信息最完整的角色） |
| 铁三角三人的身份和分工 | — |
| 所有亲手行凶者的名单 | — |
| 自己的掠食者本能和提取方式 | — |
| 苏晚救他的全部细节 | 苏晚已经遗忘灭门夜 |
| 陆岑在查悬案 | 陆岑已经发现悬案与铁三角名单重叠 |
| 赵磊是保安之子 | 赵磊接到伤害苏晚的命令（直到事后） |

### 高天禄的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 灭门案是自己授意的 | 沈执还活着 |
| 秦九鼎出钱、刀疤荣动手 | 有人在清算他的手下 |
| 陆岑是当年被打压民警的儿子 | 悬案与自己的人有关 |
| 手下在不断失踪 | 失踪原因是外部清算 |

### 秦九鼎的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 灭门案是自己出钱的 | 沈执还活着 |
| 赵父是目击者（已封口） | 赵磊已投靠刀疤荣 |
| 手下在不断失踪 | 失踪原因是外部清算 |

### 刀疤荣的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 灭门案是自己带人动手的 | 沈执还活着 |
| 打残了保安赵父 | 赵磊就是那个保安的儿子（存疑，见下方注释） |
| 手下在不断失踪 | 失踪原因是外部清算 |
| 赵磊是自己的白手套 | 赵磊的真实动机是为父治病 |

> **注释：** 刀疤荣是否知道赵磊是当年保安之子，可根据写作需要调整。如果不知道，增加悲剧层次；如果知道但不在意，增加冷血层次。

### 赵磊的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 父亲被刀疤荣打残 | 沈执是清算者 |
| 灭门案与苏晚有关（后期查到） | 沈执是灭门案受害者之子 |
| 沈执是法医专家（公开身份） | 自己已进入沈执清算名单 |
| 铁三角内部运作 | 失踪手下的真实死因 |

### 苏晚的信息状态

| 她知道 | 她不知道 |
|--------|---------|
| 父母遗命要求照顾沈执 | 沈执是掠食者 |
| 与沈执契约结婚 | 自己的异化原因 |
| 怀孕了 | 胎儿携带掠食者基因 |
| 沈执冷漠、内向、有创伤 | 沈执深夜外出是在清算 |
| — | 自己幼年曾在灭门夜救过沈执 |

### 陆岑的信息状态

| 他知道 | 他不知道 |
|--------|---------|
| 父亲查灭门案被打压致死 | 沈执是灭门案受害者之子 |
| 20年微创失踪悬案系列存在 | 沈执就是悬案制造者 |
| 悬案受害者全是暴力犯罪者 | 悬案与铁三角的深层关联（后期发现） |
| 沈执是专业法医专家 | 沈执的掠食者身份 |
| 铁三角涉黑（调查中） | 铁三角与灭门案的关系（后期发现） |
