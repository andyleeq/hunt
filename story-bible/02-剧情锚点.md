## 第一幕：骨忆 · 婚裂（第1-6集）

> **幕旨：** 所有棋子就位，灭门案伏笔层层浮现，沈执的双重生活平稳运转，铁三角的手下开始"莫名失踪"。

---

### 第1集「骨语」

**核心锚点：** 沈执以法医身份登场，建立公开人设

| 维度 | 内容 |
|------|------|
| 主线事件 | 沈执以法医人类学副教授身份出场，在一具无名白骨案中展现超凡专业能力 |
| 暗线推进 | 白骨身份指向铁三角某底层打手的旧案——这是沈执早年清算的痕迹 |
| 角色登场 | 沈执、陆岑（两人首次在案发现场相遇，建立"专家-刑警"信任关系） |
| 铁三角动态 | 高天禄在市政会议上推进老城区改造；铁三角日常运转，无异常感知 |
| 伏笔 | 沈执验骨时目光短暂异常——掠食者本能的微小闪现 |
| 集末钩子 | 骨骸身份确认：失踪者系刀疤荣手下某打手，失踪时间=20年前 |

## 第三幕：孕变 · 执序（第13-17集）

> **幕旨：** 苏晚怀孕曝光引爆一切，沈执面临秩序铁律与恩情的终极冲突，赵磊成为最终"亲手作恶者"。

---

### 第13集「孕变」

**核心锚点：** 苏晚怀孕事实曝光

| 维度 | 内容 |
|------|------|
| 主线事件 | 苏晚在一次突然晕倒后被送医，检查结果：怀孕。沈执得知消息时凝固了三秒 |
| 暗线推进 | 怀孕意味着胎儿携带掠食者基因——苏晚的异化将不可逆地加速 |
| 角色展开 | 沈执第一次面临秩序冲突：按铁律，苏晚的异化可能让她成为"需要被清除的变量"；但恩情铁律要求他保护苏晚 |
| 铁三角动态 | 赵磊接到"处理苏晚"命令后，发现苏晚怀孕——他犹豫了 |
| 伏笔 | 沈执的手写笔记中有一页：掠食者与人类繁殖，成功率极低，幸存母体将被永久改变 |
| 集末钩子 | 沈执站在苏晚病房门外，手贴着门——他在做一个从未做过的事：犹豫 |

| 伏笔 | 陆岑注意到沈执最近状态有微妙变化——从"绝对冷静"变成"偶尔走神" |

### 第15集「亲手」

| 维度 | 内容 |
|------|------|
| 主线事件 | 赵磊执行铁三角命令，对苏晚实施伤害（未致死，但造成严重后果） |
| 暗线推进 | 沈执不在场——他在执行另一次清算任务。回来后发现苏晚受伤 |

| 铁三角动态 | 刀疤荣确认赵磊已执行任务，松了口气 |
| 伏笔 | 沈执看到苏晚的伤，没有愤怒——他的眼神只有一种东西：秩序激活 |
| 集末钩子 | 沈执开始查——不是查谁伤了苏晚，而是查"这个人是否亲手动手的" |

## 第四幕：古序 · 人噬（第18集·终局）

> **幕旨：** 所有线索在最后一集交汇又散开——真相无限接近却永远无法抵达。

---

### 第18集「人噬」

**核心锚点：** 终局三重反转——正义失败、秩序继续、主题升华

| 维度 | 内容 |
|------|------|
