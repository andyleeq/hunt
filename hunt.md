## 一、故事核心主题

天生守序掠食者**沈执**，以绝对隐秘的方式清算「政商黑铁三角」手下所有**亲手犯下极刑重罪**的恶徒。

全员命运因沈执父母灭门案死死纠缠——那是铁三角人生第一次联手作恶，也是沈执秩序宿命的起点。

**铁三角自始至终完全不知道沈执的存在，更不知道有一个"隐秘清算者"在为当年那一夜闭环因果。**

### 主题内核

> 最恐怖的不是沈执的食人，是铁三角在光天化日下的阶级食人——他们永远站在法律和权力的保护伞下，永远不受清算。

## 二、底层设定

### 2.1 已删除的旧设定（不再使用）

- ~~无痕处理技术~~
- ~~沈执与铁三角合作~~
- ~~铁三角知道沈执~~

### 2.2 沈执·核心物种设定

| 属性 | 设定 |
|------|------|
| **物种** | 守序掠食智人（外表与常人无异） |
| **存活机制** | 需定期提取「清醒绝望者的松果体激素」 |
| **提取方式** | 极微创、无血、无痛苦；经特殊穴位进针，仅留无法溯源的微小针点 |
| **提取后果** | 目标在数小时内器官衰竭死亡，死因呈现为"不明原因猝死"或直接失踪 |
| **现场特征** | 无凶器、无血迹、无挣扎痕迹、无指向沈执的任何证据 |
| **公开身份** | 法医人类学副教授（某大学），官方特聘法医顾问 |
| **行动原则** | 独猎独行，绝不暴露；铁三角连他的名字都没听过 |

### 2.3 沈执·秩序铁律

1. **受恩必还** —— 对苏家救命之恩，以命相守；
2. **亲手作恶者必清** —— 只清算亲手行凶的执行者，不清算授意者、出资者；
3. **绝对隐秘** —— 不与铁三角发生任何接触、交易、合作，不主动暴露。

> **[⚠️ 逻辑待确认]** 刀疤荣在灭门夜"亲手带人动手"且"亲手把保安打至终身瘫痪"，按铁律第2条他属于"亲手作恶者"。沈执不清算他的原因需要补充说明——是因为刀疤荣是顶层目标要留到最后？还是需要调整铁律措辞为"只清算底层亲手执行者"？

### 2.4 唯一命运原点

```
沈执父母灭门案 ＝ 高天禄 × 秦九鼎 × 刀疤荣 第一次联手犯罪 ＝ 铁三角正式诞生
```

没有这件事，就没有后来的所有人。这一夜同时产生了：
- 沈执的秩序宿命
- 高天禄的仕途起点
- 秦九鼎的第一桶金
- 刀疤荣的黑道地位
- 赵磊家的破碎
- 苏晚与沈执的命运绑定
- 陆岑家的悲剧

### 2.5 铁三角认知屏障（贯穿全剧）

铁三角将手下失踪自动归因为日常黑道风险，从不怀疑存在第三方清算者：

| 角色 | 对失踪事件的认知 |
|------|------------------|
| **高天禄** | 认为是内部火并、陆岑在扫黑 |
| **秦九鼎** | 认为是黑产内讧、商业报复 |
| **刀疤荣** | 认为是仇家寻仇、手下背叛 |

### 2.6 苏晚·异化机制

| 阶段 | 表现 |
|------|------|
| **婚后初期** | 无异常，正常人类女性 |
| **怀孕早期** | 味觉改变、体温异常、夜间感知力增强 |
| **怀孕中期** | 胎儿携带掠食者基因，通过胎盘逆向改写母体基因表达 |
| **异化巅峰** | 感官、体能、情绪控制出现非人类特征 |

> **[⚠️ 逻辑待确认]** 异化是否可逆？分娩后苏晚恢复正常还是永久改变？这直接影响结局走向。

## 四、全员成长线·深度互锁

### 4.1 沈执 ｜ 隐秘秩序执行者

**背景：** 父母是工厂资料室管理员，手握厂区改制黑料，挡住铁三角升官/发财/上位之路。

**命运时间线：**

| 年龄 | 事件 |
|------|------|
| **10岁** | 高天禄授意、秦九鼎出钱、刀疤荣带人动手——父母被灭门 |
| **灭门夜** | 被苏晚父母藏进衣柜救下，幼年苏晚用身体挡住柜门 |
| **少年期** | 掠食者本能觉醒，确立「受恩必还、亲手作恶必清」的绝对秩序 |
| **成年** | 考取法医人类学副教授，用公开身份合法搜集铁三角罪证，精准锁定每一个"亲手行凶者" |
| **秘密行动期** | 独自猎杀、独自撤离、绝不暴露，启动「因果闭环秩序计划」 |

**纠缠关系：**
- 人生被毁 → 高 + 秦 + 刀联手造成
- 活命之恩 → 苏家
- 清算目标 → 铁三角所有亲手作恶的手下
- 查案同行者 → 陆岑（陆查悬案 / 沈查铁三角，两条线平行但不相交）

---

### 4.2 高天禄 ｜ 理想黑化的副市长

**背景：** 深山贫苦出身，靠乡亲接济长大，年轻时真心想做出政绩。

**命运时间线：**

| 阶段 | 事件 |
|------|------|
| **年轻科员** | 为人生第一个大政绩（厂区改制），第一次突破底线——授意灭口沈执父母 |
| **灭门案后** | 靠秦九鼎的钱、刀疤荣的暴力扫清障碍，一路升至政法副市长 |
| **掌权后** | 确实做了城建、扶贫、老城区改造的实事，但每一步官运都踩着沈家血 |
| **压案时期** | 打压主办民警陆岑父亲，导致陆父抑郁早逝 |

**核心矛盾：** 他不是纯粹的恶人——他用权力做了真实的好事，但权力本身是用灭门换来的。观众该恨他还是理解他？

**纠缠关系：**
- 仕途起点 → 灭沈执家门
- 权力根基 → 绑定秦九鼎资本 + 刀疤荣暴力
- 一生对手 → 陆岑（为父报仇的执念）

---

### 4.3 秦九鼎 ｜ 乡土资本寡头

**背景：** 底层小商贩出身，白手起家，对家乡有真感情。

**命运时间线：**

| 阶段 | 事件 |
|------|------|
| **年轻包工头** | 觊觎沈执父母手里的厂区地块，出钱雇凶灭门 |
| **灭门案后** | 拿到第一块黄金地，赚到人生第一桶黑金，建起资本帝国 |
| **发家后** | 真捐学校、修村路、资助贫困生——但原罪永远绑定沈家灭门案 |
| **封口行动** | 给赵磊父亲（灭门案目击保安）小钱封口，默许刀疤荣把人打残 |

**核心矛盾：** 他对乡亲的慷慨是真的，但这份慷慨的本钱是一家人的命。

**纠缠关系：**
- 第一桶金 → 沈执父母的命
- 资本安全 → 靠高天禄权力 + 刀疤荣暴力

### 4.4 刀疤荣 ｜ 江湖情义黑教父

**背景：** 孤儿，带年幼妹妹流浪长大，底线是护亲人、护兄弟。

**命运时间线：**

| 阶段 | 事件 |
|------|------|
| **街头混混** | 为第一次上位机会，带人执行沈执父母灭门行动 |
| **灭门夜** | 把阻止行凶的保安赵磊父亲打至终身瘫痪 |

| **掌权后** | 真护兄弟、宠妹妹、资助孤儿院——但地位全靠沈家血换来 |

**核心矛盾：** 他真的爱自己人，但他对"自己人"之外的生命毫不在意。

**纠缠关系：**
- 黑道地位 → 沈执父母的命
- 血债 → 沈执、赵磊一家

> **[⚠️ 逻辑待确认]** 刀疤荣"亲手带人动手"灭门、"亲手"打残保安，是否满足沈执"亲手作恶者必清"的铁律？如果满足，沈执为何迟迟不动他？如果不满足，需要重新措辞灭门夜的描述（例如改为"指挥手下动手，自己没沾血"）。

## 五、逻辑闭环自检

### Q1：铁三角为什么始终不知道沈执？

沈执只猎杀底层执行者，不碰顶层，不留个人痕迹，从不在铁三角的社交圈出现。铁三角手下的失踪频率足够低（20年间分散发生），且每次都有合理的"黑道日常"解释（内讧、跑路、被警方抓、仇家寻仇），铁三角没有动机去寻找一个"统一的幕后清算者"。

### Q2：没有无痕技术，为什么查不到沈执？

- **物理证据极少：** 仅一个无法溯源的微小针点，无凶器、无血迹、无挣扎
- **死因模糊：** 受害者呈现"不明原因器官衰竭"或直接失踪
- **案件分散：** 跨越20年、多个辖区，难以串联
- **沈执的身份掩护：** 法医副教授 + 官方顾问，零犯罪记录、零嫌疑

### Q3：沈执为什么不直接杀铁三角顶层？

秩序铁律：只清算「亲手行凶者」。高天禄（授意）、秦九鼎（出资）不在猎杀范围内。

> **[⚠️ 注意]** 刀疤荣的情况需要额外说明（见 2.3 和 4.4 的待确认标注）。

### Q4：陆岑为什么不怀疑沈执？

- 沈执是官方特聘法医专家，专业、冷静、无犯罪记录、无动机
- 沈执主动协助陆岑破案（选择性协助，不触及自己的清算行动）
- 陆岑的认知偏差：越熟悉的人越不会被怀疑

### Q5：全员命运是否真正彻底纠缠？

所有人的人生起点、上升、罪恶、悲剧、恩情，全部来自同一件事：沈执父母灭门案。没有一个角色是孤立的、可以被替换的。

### Q6：陆岑为什么没发现"微创失踪案"受害者都是铁三角手下？

> **[⚠️ 逻辑待确认]** 这是当前最大的潜在漏洞。建议方案：
> - **方案A：** 沈执刻意在铁三角手下之外也制造少量猎杀（为存活需要），稀释了关联性
> - **方案B：** 铁三角手下身份隐蔽，陆岑尚未查明失踪者与铁三角的从属关系
> - **方案C：** 陆岑其实已经隐约发现了重叠，这正是第13-17集的推进动力
